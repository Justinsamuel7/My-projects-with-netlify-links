Social Media Comments

The Social Media Comment Retrieval Application is a basic web project built using JavaScript and the DOM. It allows users to view comments for a specific post by making API calls to retrieve the comments data. The application displays the post titles and authors, and when users click on "Comments," it fetches and dynamically displays the corresponding comments below the post. The project showcases how to handle API requests and update the DOM to create a simple social media-like experience for users.
